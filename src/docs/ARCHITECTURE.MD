## Database Schema Overview

### Core Entities

**Users**
- id (UUID)
- email (Unique)
- passwordHash
- displayName
- role (User, Editor, Admin)
- avatar
- preferences (JSON)
- createdAt, updatedAt

**Laureates**
- id (UUID)
- firstName, lastName
- birthDate, deathDate
- birthPlace, nationality
- biography (Text)
- photo (URL)
- website (URL)
- verified (Boolean)

**Nobel Prizes**
- id (UUID)
- year (Integer)
- category (Physics, Chemistry, Medicine, Literature, Peace, Economic Sciences)
- laureate_id (Foreign Key)
- motivation (Text)
- prize_amount (Decimal)
- prize_date (Date)

**Lectures**
- id (UUID)
- title
- speaker_id (Foreign Key to Laureates)
- videoUrl (CDN URL)
- transcriptUrl
- duration (Seconds)
- category
- description
- viewCount
- uploadDate

**Research Papers**
- id (UUID)
- title
- authors (Array)
- abstract
- pdfUrl
- doi
- publicationDate
- category
- citations (Count)
- laureate_id (Foreign Key)

**Collections**
- id (UUID)
- userId (Foreign Key)
- title
- description
- isPrivate (Boolean)
- createdAt, updatedAt

**User Activity**
- id (UUID)
- userId (Foreign Key)
- activityType (view, search, download, rate)
- contentId
- contentType (laureate, lecture, paper)
- timestamp

## Technology Stack

### Frontend
- **Framework:** Next.js 14+ (React 18+)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Data Fetching:** React Query
- **Video Player:** HLS.js
- **Charts:** Recharts, D3.js
- **Forms:** React Hook Form
- **Testing:** Jest, React Testing Library

### Backend
- **Runtime:** Node.js 18+ LTS
- **Framework:** Express.js or Fastify
- **Language:** TypeScript
- **Database:** PostgreSQL 14+
- **Search:** Elasticsearch 8+
- **Cache:** Redis 7+
- **Queue:** RabbitMQ
- **API Docs:** OpenAPI 3.0
- **Testing:** Jest, Supertest

### Infrastructure
- **Cloud:** AWS
- **Container:** Docker
- **Orchestration:** Kubernetes (EKS)
- **CI/CD:** GitHub Actions
- **Monitoring:** Prometheus + Grafana
- **Logging:** ELK Stack
